One-time Setup
You only need to do this once. After setup, you can upload manga chapters directly from this page with no coding.
1
Go to GitHub Token page
Click this link → github.com/settings/tokens/new
Sign in if needed.
2
Fill in the form
• Note: type anything, e.g. manga upload
• Expiration: choose No expiration
• Scopes: tick only repo
3
Click "Generate token" and copy it
It starts with ghp_. Copy it immediately — GitHub only shows it once!
4
Paste it below
